Holiday Maker

The final project in the course "Ämnesövergripande projekt inom systemutveckling, fortsättning", a group assignment where we would create an application where users can search and book accommodations. The application is built in UWP and with the Model-View-ViewModel (MVVM) design pattern. The server application and the database are built with Entity Framework Core and ASP.NET Core.
